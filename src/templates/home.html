Harry Potter Dueling Simulator


The website generates a dueling tournament using Harry Potter characters. To start the tournament press "Start" below. Then select the number of rounds you would like to have in the tournament, 2 to 5 rounds. Once this selection is made the tournament will be created and the first round duels will be shown, where you can then select what character you think will win the tournament. The first round will then be played where one character wins each duel based on their power, intelligence, skills, etc. Each character will use basic spells that can stun their opponent, while each character may also use a bonus spell which could increase their overall score. The character with the highest score will win the duel and move on to the next round and duel again. Each round and their according results will be prompted by you. This will continue until there is only one character left, the winner.


To begin the simulator and select the number of rounds press, the "Start" button below.
